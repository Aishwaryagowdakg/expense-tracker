# Expense Tracker Application

A Python Flask application for tracking expenses and savings goals with budget alerts.

## Features
- Log daily expenses with categories
- Set monthly budgets per category
- Alerts when exceeding budget or when budget is low
- Monthly spending reports
- Budget vs actual comparison
- Basic expense sharing functionality
